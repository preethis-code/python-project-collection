# 📂 File Organizer (Python)

 A beginner-friendly Python project that automatically organizes files into neat folders.

---

## ✨ Features

* Sorts files by type (Images, Documents, Videos, etc.)
* Automatically creates folders
* Simple and easy to use

---

## 🛠️ Technologies Used

* Python
* os module
* shutil module

---

## ▶️ How to Run

1. Create a folder named `test_folder`
2. Add some random files inside it
3. Update the folder path in `main.py`
4. Run:
   python main.py

---

## 📌 Example

**Before:**
test_folder/
file1.jpg
doc1.pdf
song.mp3

**After:**
Images/
Documents/
Music/
